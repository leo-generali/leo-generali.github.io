Guess the Color!

HTML, Skeleton, JavaScript

Players are given a random RGB value and must guess which of the squares is that color. A variable score is added to the player's point total depending on how many attempts the player took to find the right color. The game is won once the player reaches 300 points. DOM manipulation is handled through vanilla Javascript.
