Color Guessing Game

HTML, Skeleton, JavaScript

Project based off of the Udemy Web Developer Bootcamp. The player is given a random RGB value and must guess which of the squares is that color. Players have the option between selecting an easy mode, or a hard mode. A variable score is added to the player's point total depending on how many attempts the player took to find the right color. Once the player scores 300 points, the game is over and they have won.

This game uses vanilla JavaScript for all of it's DOM manipulation. I challenged myself to not use frameworks for this project.
