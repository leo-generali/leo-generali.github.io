Metro Map

HTML, CSS, Knockout.js

Single page, responsive DC Metro map application. Users can either query through all stations through a search bar, or click on the data points to receive incoming transit information. AJAX requests are used to pull information from WMATA and District of Columbia Open Data APIs.

Running Pace Calculator

HTML, CSS, React

Created a single-page application designed with running athletes in mind. Built with React and Flexbox, this tool lets users quickly find the pace of their latest run. Users can switch between metric and imperial measurement systems at the press of a button.
